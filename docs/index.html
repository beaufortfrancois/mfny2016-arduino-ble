Chrome Web Bluetooth Example. Requires Chrome 53 on Mac, Linux, ChromeOS, or Android. (Sorry, doesn't work on Windows or iOS.) This code has a token that lets it run on https://don.github.io without any browser flags. If you run this on a different URL, including localhost, get your own key or enable web bluetooth from chrome://flags/#enable-web-bluetooth
